Top 5 critical gaps

  • SL Wait unit mismatch. Spec says seconds (SL Wait (sec)). Code uses bars (ExitConfig.sl_wait_bars, models.py:38). On a 1-min bar, "30" means 30 minutes, not 30 seconds — 60× off.
  • D2 "Same Contract" rename not propagated. Code constant in backtest_runner.py:722 and UI tab options in stoploss.json:14, target.json:14 still expose "ReExecute SameStrike at EntryPrice". D2 is marked Locked.
  • Leg-level "Execute (other leg by leg_id)" missing. Spec marks ✅ for all asset classes; no leg_id cross-reference exists in ExitConfig or action routing.
  • Leg-level "ReEntry (price-wait)" missing. Spec marks ✅; only re_execute (immediate re-arm) is implemented. models.py:249 docstring acknowledges the gap.
  • Leg-level "KeepLegRunning" missing. Spec marks ✅; no equivalent action enum or flag exists.
